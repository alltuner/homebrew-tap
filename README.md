# Homebrew Tap for Factory Floor

This is the official [Homebrew](https://brew.sh) tap for [Factory Floor](https://factory-floor.com).

## Installation

```bash
brew install --cask alltuner/tap/factoryfloor
```

## Updating

```bash
brew upgrade --cask factoryfloor
```
